# SendingScheduler
Проект кор нугетов для быстрой развертки проектов серии Sending  для обязательной отправки данных в основном внешним интеграторам.
В отличие от старого проектта Sending эвенты не содержат в себе отправляемое тело запросов. Для этой роли используется инстанс Redis, который способен держать в памяти до 500 мб тела.
Пакет состоит из базового нугета SendingScheduler.Core и трех зависимых от него пакетов SendingScheduler.Queue SendingScheduler.Results SendingScheduler.Send
Примеры с использованием этих зависимостей имеют суффикс .Example
